# Comparação de AutoGluon e H2O AutoML no Dataset de Diabetes

Este repositório contém o código e os experimentos de um trabalho acadêmico que compara o desempenho dos frameworks **AutoGluon** e **H2O AutoML** na tarefa de **classificação de diabetes** a partir de variáveis clínicas (ex.: glicose plasmática, IMC, idade etc.).
